🚗 RealCarService

Sistema web desenvolvido para apresentação de serviços automotivos, com foco em oficinas mecânicas e atendimento ao cliente. O projeto possui interface moderna, responsiva e intuitiva, permitindo que usuários conheçam os serviços oferecidos e entrem em contato facilmente.

📋 Sobre o Projeto

O RealCarService foi desenvolvido com o objetivo de criar uma landing page profissional para oficinas mecânicas, proporcionando uma experiência simples e eficiente para os clientes.

A aplicação apresenta informações sobre a empresa, serviços disponíveis e canais de contato, facilitando a comunicação entre clientes e oficina.

✨ Funcionalidades

✅ Página inicial moderna e responsiva

✅ Seção institucional (Sobre a empresa)

✅ Apresentação dos serviços oferecidos

✅ Formulário de contato

✅ Integração com WhatsApp

✅ Integração com E-mail

✅ Layout adaptado para dispositivos móveis

✅ Navegação intuitiva
